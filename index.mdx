## What this API does

RM BG API lets you authenticate users, remove image backgrounds, manage tokens, and process payments.

## Start here

1. Read [/guides/authentication](/guides/authentication).
2. Get a JWT token from an auth endpoint.
3. Use the JWT token in the `token` header for protected endpoints.
4. Issue a managed token before calling `/background/remove/*` endpoints.

- Authentication and tokens: [/guides/authentication](/guides/authentication)

- Payments: [/guides/payments](/guides/payments)

## Что делает этот API

RM BG API позволяет авторизовать пользователей, удалять фон изображений, управлять токенами и проводить платежи.

## С чего начать

1. Прочитайте [/guides/authentication](/guides/authentication).
2. Получите JWT-токен через один из auth-эндпоинтов.
3. Передавайте JWT-токен в заголовке `token` для защищенных эндпоинтов.
4. Перед вызовом `/background/remove/*` выпустите managed-токен.
