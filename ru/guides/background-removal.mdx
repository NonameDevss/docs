## Предусловие

Перед вызовом background-эндпоинтов пройдите [Авторизацию и поток токенов](/ru/guides/authentication).

## Поддерживаемые эндпоинты

- `POST /api/v1/background/remove/url`
- `POST /api/v1/background/remove/base64`
- `POST /api/v1/background/remove/multipart`

Для всех трех эндпоинтов требуется:

- Заголовок `Authorization: <managed_token>`

## Пример: удаление фона по URL

```bash
curl -X POST 'https://api.cutbg.net/api/v1/background/remove/url' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: <managed_token>' \
  -d '{
    "image_url": "https://example.com/photo.jpg"
  }'
```

Пример ответа:

```json
{
  "image_base64": "iVBORw0KGgoAAAANSUhEUg..."
}
```

## Декодирование `image_base64` в памяти

Возьмите `image_base64` из ответа API и декодируйте его в объект изображения в памяти.

    ```python
    import base64
    from io import BytesIO
    from PIL import Image

    image_base64 = response_json["image_base64"]
    image_bytes = base64.b64decode(image_base64)
    image = Image.open(BytesIO(image_bytes))  # PIL image in memory
    ```

  <Tab title="Java">
    ```java
    import java.util.Base64;
    import java.io.ByteArrayInputStream;
    import java.awt.image.BufferedImage;
    import javax.imageio.ImageIO;

    String imageBase64 = responseJson.get("image_base64").asText();
    byte[] imageBytes = Base64.getDecoder().decode(imageBase64);
    BufferedImage image = ImageIO.read(new ByteArrayInputStream(imageBytes)); // in memory
    ```

    ```csharp
    using var image = Image.FromStream(stream); // in-memory image
    ```
