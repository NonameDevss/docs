## Флоу оплаты подписки

1. Получите планы: `GET /api/v1/payments/options`.
2. Создайте ссылку на оплату: `POST /api/v1/payments/link`.
3. Проверьте список инвойсов: `GET /api/v1/payments/invoices`.
4. Получите детали инвойса: `GET /api/v1/payments/invoices/{invoice_id}`.

Для всех эндпоинтов, кроме `GET /payments/options`, требуется:

- Заголовок `token: <jwt_token>`

## Пример: создание ссылки на оплату

```bash
curl -X POST 'http://localhost:8000/api/v1/payments/link' \
  -H 'Content-Type: application/json' \
  -H 'token: <jwt_token>' \
  -d '{
    "plan": "month"
  }'
```
