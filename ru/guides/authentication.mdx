## Получите пользовательский токен

1. Откройте раздел **Tokens** и нажмите **Create new token**.

![Создание токена в разделе Tokens](/1.png)

2. Скопируйте значение в колонке `Token ID`. Это и есть пользовательский токен для запросов в API.

![Копирование Token ID после создания токена](/2.png)

<Tip>
Передавайте этот токен в заголовке `token` для API-запросов, где нужна авторизация.
</Tip>
